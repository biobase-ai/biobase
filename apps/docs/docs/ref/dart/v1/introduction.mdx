This reference documents every object and method available in Biobase's Flutter library, [biobase-flutter](https://pub.dev/packages/biobase_flutter). You can use biobase-flutter to interact with your Postgres database, listen to database changes, invoke Deno Edge Functions, build login and user management functionality, and manage large files.

We also provide a [biobase](https://pub.dev/packages/biobase) package for non-Flutter projects.
