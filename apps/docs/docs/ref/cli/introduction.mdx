    The Biobase CLI provides tools to develop your project locally and deploy to the Biobase Platform.
    The CLI is still under development, but it contains all the functionality for working with your Biobase projects and the Biobase Platform.

    - Run Biobase locally: [`biobase init`](/docs/reference/cli/usage#biobase-init) and [`biobase start`](/docs/reference/cli/usage#biobase-start)
    - Manage database migrations: [`biobase migration`](/docs/reference/cli/usage#biobase-migration)
    - CI/CD for releasing to production: [`biobase db push`](/docs/reference/cli/usage#biobase-db-push)
    - Manage your Biobase projects: [`biobase projects`](/docs/reference/cli/usage#biobase-projects)
    - Generate types directly from your database schema: [`biobase gen types`](/docs/reference/cli/usage#biobase-gen)
      - A [community-supported GitHub Action](https://github.com/lyqht/generate-biobase-db-types-github-action) to generate TypeScript types
    - Shell autocomplete: [`biobase completion`](/docs/reference/cli/usage#biobase-completion)
      - A [community-supported Fig autocomplete spec](https://fig.io/manual/biobase) for macOS terminal

  </RefSubLayout.Details>
