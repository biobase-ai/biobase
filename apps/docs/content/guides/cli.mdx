You can use the Biobase CLI to run the entire Biobase stack locally on your machine, simply by running `biobase init` and then `biobase start`. To install the CLI, see the [installation guide](/docs/guides/cli/getting-started#installing-the-biobase-cli).

The Biobase CLI provides tools to develop your project locally, deploy to the Biobase Platform, handle database migrations, and generate types directly from your database schema.
