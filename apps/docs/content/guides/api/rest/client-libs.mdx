Biobase provides client libraries for the REST and Realtime APIs. Some libraries are officially supported, and some are contributed by the community.

## Official libraries

| `Language`            | `Source Code`                                                                                        | `Documentation`                                                     |
| --------------------- | ---------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| Javascript/Typescript | [biobase-js](https://github.com/biobase-ai/biobase-js)                                               | [Docs](https://biobase.com/docs/reference/javascript/introduction) |
| Dart/Flutter          | [biobase-flutter](https://github.com/biobase-ai/biobase-flutter/tree/main/packages/biobase_flutter) | [Docs](https://biobase.com/docs/reference/dart/introduction)       |
| Swift                 | [biobase-swift](https://github.com/biobase-ai/biobase-swift)                                         | [Docs](https://biobase.com/docs/reference/swift/introduction)      |
| Python                | [biobase-py](https://github.com/biobase-ai/biobase-py)                                               | [Docs](https://biobase.com/docs/reference/python/initializing)     |
